# Тест-кейси програмної системи «Готель»

| ID | Назва | Передумова | Основні кроки | Очікуваний результат |
|---|---|---|---|---|
| TC-01 | View rooms list | Система відкрита у браузері | Відкрити головну сторінку, перейти до списку номерів, перевірити картки номерів | Список номерів відображається коректно |
| TC-02 | Create booking request | Відкрита форма бронювання | Заповнити дані клієнта, дати заїзду/виїзду, номер кімнати, натиснути кнопку бронювання | Система створює заявку на бронювання |
| TC-03 | Validate input data | Відкрита форма бронювання | Залишити обов'язкові поля порожніми, натиснути кнопку бронювання | Система не створює заявку без обов'язкових даних |
| TC-04 | Save booking request | Форма заповнена коректними даними | Створити заявку і перейти до списку бронювань | Створена заявка відображається у списку бронювань |
| TC-05 | Search and filter rooms | Список номерів завантажено | Ввести тип номера, вибрати статус, натиснути пошук | Система показує номери, що відповідають умовам пошуку |
| TC-06 | Browser compatibility and usability | Файли системи доступні | Відкрити систему у браузері, перевірити секції, кнопки та поля | Система відкривається коректно, інтерфейс зрозумілий |
