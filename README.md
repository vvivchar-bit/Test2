# Практичне заняття №12 — Тестування програмної системи «Готель» у ReQtest

## Опис роботи

У цьому репозиторії розміщено матеріали до практичного заняття №12 з тестування програмної системи **«Готель»**.

Метою роботи є виконання тестування основних функцій навчального вебпрототипу системи «Готель» за допомогою середовища **ReQtest** та підтвердження результатів тестування скріншотами роботи системи у браузері.

## Що виконано

- створено тестовий запуск **TR-02 System-Hotel functional testing** у ReQtest;
- додано 6 тест-кейсів для перевірки основних функцій системи;
- виконано тестування та отримано результат **OK** для всіх тест-кейсів;
- перевірено роботу прототипу системи у браузері;
- підготовлено DOCX-звіт.
